Arquitectura de Red

Tu negocio, tu red,
tus datos.

ELRHOS opera en arquitectura Cliente-Servidor sobre red local (LAN). Tus datos nunca salen de tu negocio, nunca pagan mensualidad de nube, y funcionan aunque se caiga el internet.

🖥

PC Servidor

Una computadora actúa como servidor central. Aloja la base de datos y el motor de ELRHOS. Requiere Windows 10/11 x64.

Motor DBAPI localWin x64
💻

Terminales Cliente

El resto de las PCs del negocio instalan solo el cliente ligero de ELRHOS y se conectan al servidor por IP local. Sin base de datos propia.

Cliente ligeroTCP/IP
🔒

Datos 100% locales

Ningún dato de tu negocio sale a servidores externos. La base de datos vive en tu PC servidor — tú tienes el control total.

On-premiseSin nubePrivacidad
📡

Sin internet requerido

Opera completamente sin conexión a internet. Ideal para zonas con servicio inestable o negocios que prefieren mantener su información offline.

Offline-firstSin mensualidad
🔄

Sincronización en tiempo real

Las ventas, movimientos de inventario y cambios de precio se reflejan en todas las terminales en tiempo real, mientras operen en la misma red.

Real-timeMulti-caja

Aplicación de escritorio

Python + PyQt5

La aplicación principal está construida en Python 3.11+ usando PyQt5 para la interfaz gráfica nativa. Esto garantiza rendimiento de aplicación de escritorio real, sin los overhead de Electron ni de navegadores embebidos.

Python 3.11+ PyQt5 5.15+ SQLAlchemy PyInstaller Pillow python-escpos
pos_window.py 
from PyQt5.QtWidgets import QMainWindow
from core.inventory import ProductSearch
from core.sales import SaleTransaction

class POSWindow(QMainWindow):
    def search_product(self, query):
        # Búsqueda incremental desde primer char
        results = ProductSearch.find(
            query=query,
            limit=20,
            fuzzy=True
        )
        self.update_results_table(results)

    def confirm_sale(self):
        tx = SaleTransaction(
            items=self.cart.items,
            payment=self.payment_method,
            cashier=self.current_user
        )
        tx.commit()  # sync a todas las terminales

App Android

Kotlin + Jetpack

La app Android está desarrollada en Kotlin con arquitectura MVVM usando Jetpack Compose para la UI. Se comunica con el servidor ELRHOS vía API REST local en la red LAN.

Kotlin 1.9+ Jetpack Compose Retrofit 2 Room DB Hilt / DI Android 8.0+
DashboardViewModel.kt 
@HiltViewModel
class DashboardViewModel @Inject constructor(
    private val repo: SalesRepository
) : ViewModel() {

    val todaySales = repo
        .getTodaySales()
        .stateIn(
            scope = viewModelScope,
            started = SharingStarted.WhileSubscribed,
            initialValue = SalesState.Loading
        )
    // Conecta vía LAN — no internet
}
