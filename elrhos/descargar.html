Guía de instalación

Listo en 5 minutos.

1
Verifica que tu Windows sea de 64 bits

Ve a Inicio → Configuración → Sistema → Acerca de. En "Tipo de sistema" debe decir "Sistema operativo de 64 bits".

⚠️ El instalador actual NO es compatible con Windows de 32 bits (x86). Esa versión está en desarrollo.
2
Descarga el instalador

Haz clic en el botón de descarga. El archivo ELRHOS_Setup_v0.8.3.exe pesa aproximadamente 45 MB.

3
Ejecuta como administrador

Haz clic derecho sobre el archivo descargado y elige "Ejecutar como administrador". Esto es necesario para instalar el motor de base de datos correctamente.

Si Windows Defender SmartScreen te advierte, haz clic en "Más información" → "Ejecutar de todas formas". Esto ocurre porque el instalador aún no está firmado digitalmente (en proceso).
4
Sigue el asistente de instalación

El asistente te guiará paso a paso. Elige si instalarás como servidor (PC principal con la base de datos) o como terminal cliente (PCs adicionales).

📁 Ruta de instalación por defecto:
C:\Program Files\ELRHOS\
5
Configura el servidor (solo si es el servidor)

Al primer inicio, el asistente de configuración te pedirá el nombre del negocio, datos del administrador y la contraseña de base de datos. Las terminales cliente solo necesitan la IP del servidor.

6
¡Listo! Carga tu inventario y vende.

Usa el módulo de inventario para cargar tus productos (puedes importar desde Excel/CSV) y comienza a vender desde el mismo día de instalación.

No. ELRHOS funciona completamente offline. Solo necesitas una red local (LAN) entre tus computadoras si quieres usar multi-terminal. El sistema no requiere internet en ningún momento para operar.
La base de datos de ELRHOS usa transacciones atómicas — si el sistema falla durante una operación, la transacción se revierte automáticamente. Recomendamos activar el backup automático diario desde la configuración del servidor para mayor seguridad.
No hay límite de terminales impuesto por el software. La cantidad práctica depende de la capacidad de tu red local y del servidor. En pruebas internas funciona bien con hasta 8 terminales simultáneas en una red estándar.
